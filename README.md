# Smart Meter Guardian
Real-time, end-to-end asset tracking system for smart meter deployment.

## Features
- Real-time GPS tracking
- Guardian tag inventory
- Installer leaderboard
- Risk scoring and alerts
- Carbon impact tracking

## Tech Stack
Frontend: React + Tailwind
Backend: FastAPI  
Database: SQLite + SQLAlchemy

## Run Backend
uvicorn main:app --reload

## Run Frontend
npm run dev
